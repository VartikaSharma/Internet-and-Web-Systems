Properties

Width

It is used to define the width for an element: ex, 20px. It can also be specified in percentage. Suppose we gave 10% for width, it will span across the 10% of the total width of the containing element.

Height

It is used to define the height for an element: ex, 20px. It can also be specified in percentage. Suppose we gave 10% for height, it will span across the 10% of the total height of the containing element.

Position

Used for positioning the element. For example, giving value "fixed" to this property, will keep the element fixed, even while scrollig that element would be displayed at the same position.

Float

It is used to define whether an element must float, like float:left makes an element to be postioned to the left of the containing element.

Background-color

It is used to specify the background color of an element.

Display

It is used to define the display properties of an element, like giving value "none", will hide that particular element, "block" displays it in a block, "inline-block" displays multiple elements side by side in a row.

Border

It is used to define a border for an element. It allows to specify the thickness of the border, along with the color, and styling it using "solid", "dotted", etc.

Border-radius

It is used to define the radius for the border of an element. Mainly used if we want to style an element by having round corners. It can also be used to display an element (div, image) round.

Padding

It is used to define inner spacing from the container-elements' boundary.

Margin

It is used to define the outer spacing of the element from other elements. Ex, giving a margin of 5px will cause that element to have a gap of 5px on all sides outside from every other element.

Text-align

It is used to specify the positioning of the text inside an element. It can be given values like centre, bottom, top.

Font-size

It is used to define the size of the text in that element.

Color

It is used to give color to the text in that element.

Text-decoration

It is used to specify the decoration that is to be added/removing any default decoration, to the text. For example, displaying a text with a line through it, can be done using this property.

Font-family

It is used to style the text by specifying the font for it.

Top

It is used to specify the top edge gap of a positioned element. Giving a negative value will drag that element towards the top and a positive value drags it away from the top.

Bottom

It is used to specify the bottom edge gap of a positioned element.Giving a negative value will drag that element towards the bottom and a positive value drags it away from the bottom.

Right

It is used to specify the right edge gap of a positioned element.Giving a negative value will drag that element towards the right and a positive value drags it away from the right.

Left

It is used to specify the left edge gap of a positioned element. Giving a negative value will drag that element towards the left and a positive value drags it away from the left.
